# Potrillos — Registro de Crecimiento

Una app PWA para seguimiento de etapas de crecimiento de potrillos.

## Cómo instalar en iPhone

1. Abre en Safari: https://juancgarciah.github.io/rancho-potros/
2. Toca el ícono **Compartir** (flecha hacia arriba)
3. Selecciona **Agregar a pantalla de inicio**
4. Toca **Agregar**

## Características

- 📱 Funciona sin internet
- 🐴 Registro de crecimiento por etapas
- 💾 Datos guardados localmente
- ⚙️ Etapas personalizables
